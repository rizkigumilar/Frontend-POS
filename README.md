# POS coffee store


## Prerequiste.
- React Js
- Reactstrap / Bootstrap
- npm (node module Package)
- Redux (for managemant state)
- moment 

## Installation
### Clone
```
$ git clone https://github.com/rizkigumilar/Frontend-POS.git
$ cd Frontend-POS
$ npm install
```

### Start Development Server
```
$ npm start
```
